# spamdb
This directory holds the IPv4 numbers and other information used by dnsbl.

Each octet of the IP number has its own level, e.g. information about 5.4.3.2
would be in spamdb/5/4/3/2/5.4.3.2.*

The exact structure and contents of the directories and files are subject to
change as the software is developed. Hopefully this documentation is up to
date, but as with all software, the source is the true authority.
